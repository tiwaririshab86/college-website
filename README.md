🎓 College Placement Portal
A modern and responsive College Placement Portal developed using HTML5 and CSS3. This project simulates a university placement website inspired by professional educational portals and demonstrates comprehensive front-end development concepts.<br><br>

✨ Features<br><br>
Modern and responsive UI design<br>
HTML5 semantic elements<br>
Sticky navigation bar<br>
Hero section with call-to-action button<br>
Department and placement service cards<br>
Top recruiters section<br>
Placement statistics and course tables<br>
Student registration form<br><br>

Login form<br><br>
Resume upload functionality<br>
Feedback form<br>
Progress bars and skill meters<br>
Placement process and FAQ section<br>
Image gallery<br>
Audio and video integration<br>
YouTube video embedding<br>
Google Maps integration<br>
Professional footer design<br>

🛠️ Technologies Used<br><br>
HTML5<br>
CSS3<br>
Google Fonts (Poppins)<br>
Responsive Web Design<br>

CSS Grid<br>
Flexbox<br>
Animations and Hover Effects<br>
