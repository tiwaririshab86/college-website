🎓 College Placement Portal
A modern and responsive College Placement Portal developed using HTML5 and CSS3. This project simulates a university placement website inspired by professional educational portals and demonstrates comprehensive front-end development concepts.

✨ Features
Modern and responsive UI design
HTML5 semantic elements
Sticky navigation bar
Hero section with call-to-action button
Department and placement service cards
Top recruiters section
Placement statistics and course tables
Student registration form

Login form
Resume upload functionality
Feedback form
Progress bars and skill meters
Placement process and FAQ section
Image gallery
Audio and video integration
YouTube video embedding
Google Maps integration
Professional footer design

🛠️ Technologies Used
HTML5
CSS3
Google Fonts (Poppins)
Responsive Web Design

CSS Grid
Flexbox
Animations and Hover Effects
